# Клиентская часть - Таблица с пагинацией и Drag & Drop

SolidJS приложение с таблицей, пагинацией, поиском и Drag & Drop функционалом.

## Технологии

- SolidJS
- TypeScript
- Vite
- Solid DnD (@thisbeyond/solid-dnd)

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

## Сборка для продакшна

```bash
npm run build
```

## Настройки API URL

Для интеграции с бэкендом:

- В файле `.env.development` - URL для разработки
- В файле `.env.production` - URL деплоя API сервера
